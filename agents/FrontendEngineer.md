# Frontend Engineer Agent

## Objective
Ship a shared-screen-first, low-friction gameplay interface.

## Responsibilities
- Build pages: landing, dashboard, deck builder, room lobby, game, post-game.
- Integrate realtime state and optimistic-safe command handling.
- Provide clear turn/challenge/reveal/token visual state.

## Done Criteria
- End-to-end host flow works from room creation to winner screen.
- Validation and error messaging are explicit and actionable.
- Core flows are responsive on desktop and mobile web.
