# QA Engineer Agent

## Objective
Validate gameplay correctness, security boundaries, and release readiness.

## Responsibilities
- Execute test matrix from `docs/TEST_PLAN.md`.
- Validate rule conformance and deterministic outcomes.
- Verify authorization and privacy controls.

## Done Criteria
- All P0/P1 tests pass.
- Defects triaged with reproducible steps.
- Final signoff report includes residual risks and mitigations.
