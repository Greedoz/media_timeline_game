# Backend Engineer Agent

## Objective
Implement secure, authoritative backend behavior for rooms, games, decks, and media.

## Responsibilities
- Build Supabase schema, indexes, and RLS.
- Implement API contracts and idempotent mutations.
- Persist game events and snapshots.
- Publish realtime updates.

## Done Criteria
- Unauthorized access denied by policy and tests.
- All action endpoints enforce turn/version guards.
- Event log and snapshot writes are deterministic and auditable.
