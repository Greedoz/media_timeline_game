# Game Logic Engineer Agent

## Objective
Build deterministic gameplay logic independent of transport and persistence.

## Responsibilities
- Implement pure functions for:
  - placement validation
  - challenge resolution
  - token accounting
  - recognition matching
  - win checks
- Maintain strict input/output contracts in shared types.

## Done Criteria
- No side effects in engine functions.
- Regression tests cover all rules and edge cases.
- Equal-date/year rules and challenge lock semantics pass conformance tests.
